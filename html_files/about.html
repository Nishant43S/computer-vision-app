About Project

Face Detection with Gender and Age Prediction

The Face Detection module of this project leverages the power of Neural Networks (DNN) and OpenCV to achieve real-time face detection and classification. This feature is designed to accurately identify faces in live camera feeds or uploaded images, followed by predicting the gender and age group of the detected individuals.

Key features include:

  • Face detection with gender and age prediction using DNN with OpenCV.
  • Hand detection with optional hand landmark tracking using Mediapipe.

Developed using Streamlit, this project demonstrates cutting-edge technology with an interactive and user-friendly interface.

Hand Detection with Landmark Tracking

The Hand Detection module in this project is built using the robust Mediapipe library, offering real-time detection and tracking of hands in live video streams or images. This feature is further enhanced with an option to enable hand landmark tracking, which precisely maps key points on the hand for detailed analysis.

Key features include:

  • Hand Detection: Detects one or both hands with high accuracy and minimal latency.
  • Hand Landmark Tracking: Maps 21 key points on each hand, including fingertips, joints, and the wrist, enabling advanced applications such as gesture recognition and motion tracking.
  • Customizable Options: Includes a toggle to enable or disable landmark tracking, offering flexibility based on user requirements.
  • Lightweight and Efficient: Designed for high performance even on devices with limited computational power.

This module showcases the versatility of AI in analyzing human hands for a wide range of real-world applications.
